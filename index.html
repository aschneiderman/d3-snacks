tools

  • Block Builder: great tool for creating a publicly accessible D3 example; also great for playing around with other folks' d3 creations. To use it, you'll need a Github account.
  • Served: if you're running d3 from a file on your PC and you need to read in a data file or a variety of other actions that can cause security issues, most browsers require you run the file using a web server. Served is a tiny server that easily lets you do that.
